ソフトドリンク
▶
  1. ソフトドリンク各種：グラスに氷を5つ入れ、各種ドリンクを注ぐ
  2. カルピス：氷5つ → ポンプ2回（下まで） → 水を入れてマドラーで混ぜる
この項目を理解しました
焼酎（ロック・割り）
▶
  1. ロック：ロックグラスに氷5つ → ショット1回 → マドラーで混ぜる
  2. 水割り：焼酎グラスに氷5つ → ショット1回 → 水を上1cmまで → 混ぜる
  3. ソーダ割り：焼酎グラスに氷5つ → ショット1回 → ソーダ上1cmまで → 混ぜる
  4. お湯割り：グラスをお湯で温めて捨てる → ショット1回 → お湯上1cmまで → 混ぜる
この項目を理解しました
酒カクテル・カクテル・タルハイ
▶
  1. サケジンジャー：氷5つ → ライム（25+10）→ 酒（25×2）→ ジンジャー → 軽く混ぜる
  2. サムライロック：氷5つ → ライム（25）→ 酒（25×3）→ 混ぜる
  3. タルハイ：氷5つ → タルハイ8分目 → シロップ25 → 混ぜる
  4. カシス各種：氷5つ → カシス（25×2）→ 割材を上2cmまで → よく混ぜる
注意
  1. カシスは比重が重いため、必ずしっかり混ぜる
この項目を理解しました
梅酒
▶
  1. ロック：氷5つ → ポンプ下4回 → 混ぜる
  2. 水割り・ソーダ割り：氷5つ → ポンプ下4回 → 割材上2cm → よく混ぜる
  3. お湯割り：陶器小を温める → ポンプ下4回 → お湯上1cm → 混ぜる
この項目を理解しました
ビール・ハイボール
▶
  1. 生ビール：AsahiロゴA頂点まで注ぎ、DRYロゴ下まで泡をのせる
  2. フリージングハイボール：氷4つ → 専用機で上1cmまで注ぐ
  3. びん・ノンアル：栓を開け、咲蔵グラスと提供
  4. 澪ドライ：席で開栓し、冷酒グラス（吟醸）で提供
