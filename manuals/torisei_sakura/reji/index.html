レジ締め
▶
※金庫からレジ金箱と入金袋を出しておく
  1. 画面 → 点検ボタン → 未収伝票ボタン（下） → 未収伝票が無いか確認 → 閉じるボタン
  2. 画面 → 業務メニューボタン → 点検処理ボタン → レジ金カウント（過不足±０になれば良い）
  3. 金額合えば 全種別ボタン → 日計ボタン → 印刷ボタン → レシート１枚出す。（A）
  4. （A）を見ながら、その伝票があるか、確認する。 → 合えは 閉じるボタン
    ※数値や件数の差異がある場合、原因つぶす。（大体、処理キー押しミス）
  5. レジキーボード右上の 精算ボタン → 03300 → 決定ボタン → 精算（レジ締）ボタン → OKボタン
  6. （先程の合わせた在高が反映されているので）
    日計ボタン → 印刷ボタン → ４枚レシート（出てきた順にホチキス）
    ※同時にクレジット端末にて集計（下） → クレジット（左上） → 日計 → クレジット集計を1枚出しておく
  7. パソコン下棚に緑封筒に入金額用意し、入金袋へ
    ※入金額について
    入金額はジャーナルの上位に載ってる
    ＜金種内訳集計＞
    現金売　○件→￥〇〇（ここの金額をレシート裏に日付・担当者氏名とともに記載する ）
  8. 残ってるレジ金をレジ金箱にしまい、レジ画面は 次へボタンで自動シャットダウン
  9. レジ小銭ゾーン裏返して、レジ金、金庫へしまう
  10. 終了（日報へ）
