レジ業務マニュアル

すぐ確認できるレジ操作マニュアル（店舗用）

1. 基本の会計の流れ
最も使用頻度の高い基本オペレーション
▶
  1. お客様のテーブル番号を確認する
  2. ハンディ or 注文票の内容とレジの表示が一致しているか確認
  3. 割引・クーポンがあるか確認（※提示済みか必ず確認）
  4. 金額をお客様へ読み上げる（例：”お会計 3,280円です”）
  5. お会計方法（現金・カード・QR）を確認
✔ 表示金額と注文内容の不一致が最もクレームになりやすい。必ず「注文確認 → 金額提示」の順で進行。
2. 現金会計
間違いを防ぐための標準手順
▶
  1. 金額を声に出して確認（例：「3,000円お預かりします」）
  2. レジに入力し、お釣りの金額を目視で確認
  3. トレーに「お釣り → レシート → 両手でお渡し」
  4. 最後に「ありがとうございました」と必ず目を見て挨拶
画像を入れる
お釣りの渡し方例
トレーの上で整理し、丁寧に渡す。
✔ 札をしまう前にレジへ「金額入力 → 釣り確認」を徹底 ✔ お釣りの間違いは必ず記録と報告
3. カード会計（クレジット）
サイン・暗証番号の対応含む
▶
  1. お客様へカード方式を案内（タップ・挿入・スワイプ）
  2. 支払い端末に金額を入力してカード処理
  3. 暗証番号入力 or サインが必要な場合は案内
  4. レシート控えを確認し、必要に応じてお客様へ渡す
  5. 「ありがとうございました」で完了
✔ タッチ決済の場合はカードを受け取らず、お客様自身でタップ ✔ 決済失敗は“端末のエラーコード”を必ず確認
4. QRコード決済（PayPay等）
スマホ決済の標準フロー
▶
  1. QR決済の利用を確認する
  2. レジに金額入力 → 決済画面をお客様へ提示
  3. お客様のスマホで読み取り → 支払い実行
  4. 決済完了画面を「必ず目視で確認」
  5. レシートを渡して終了
✔ 決済完了画面を見ずに処理すると“未払い”が発生するため厳禁 ✔ 店舗側QRの場合、間違った金額入力に注意
5. 伝票割り（複数人の割り勘）
最もトラブルが多い操作
▶
  1. お客様に「均等割り」か「個別会計」かを確認
  2. 均等割りの場合 → レジに人数を入力し自動計算
  3. 個別会計の場合 → 注文内容を確認しながら1人分ずつ会計
  4. 全体金額との誤差がないか必ず確認
✔ 個別会計は最も時間がかかるため、混雑時は“先に案内”する ✔ 最後の1人分が合わないケースが多いため、こまめに金額確認
6. レジ締め（閉店作業）
現金差異防止のための基本手順
▶
  1. 現金売上の合計をレジ上で確認
  2. レジ内の現金を点検し、差異がないか照合
  3. カード売上・QR売上の合計金額を確認
  4. 売上レポートを印刷（または写真保存）
  5. 金庫に現金を封筒で保管 → 店長へ報告
✔ 差異がある場合は「いつ・どの会計で起きたか」を逆算 ✔ 伝票の入力ミスが最も多いため、日中のチェックが重要
