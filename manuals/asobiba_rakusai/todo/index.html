緊急時対応マニュアル

クレーム・強盗・体調不良・事故・火災などの対応手順

1. クレーム対応
飲食店で最も発生しやすいトラブル
▶
【基本姿勢】
  1. 遮らず最後まで聞く（絶対に言い訳しない）
  2. 「申し訳ございません」とまず謝罪
  3. 事実確認 → 解決策を提示
  4. 店長へ即報告・引き継ぎ
  5. 原因を共有し再発防止へ
【よくあるクレーム】
  • 提供が遅い：状況説明 → すぐ対応 → 店長へ報告
  • 注文違い：即交換 → 再発防止を共有
  • 接客が悪い：否定しない → 店長対応へ切り替え
✔ 言い訳・反論は絶対NG ✔ 自分で判断できない場合は店長を呼ぶ
2. 強盗・不審者
身の安全を最優先に行動する
▶
【危険行為があった場合】
  1. 挑発・反抗しない（命の安全が最優先）
  2. お金や物品は抵抗せず渡す
  3. お客様・スタッフの安全確保を優先
  4. 犯人を追わない
  5. 安全確保後に110番通報
  6. 店長・責任者に連絡
✔ 追いかける・動画撮影で近づく → 絶対NG ✔ 何よりも「命」が最優先
3. 商品の入れ忘れ（店内・テイクアウト）
最も起こりやすいミス
▶
【入れ忘れが発覚したら】
  1. まず事実確認（伝票と提供内容を照合）
  2. 不足商品を最優先で作り直す
  3. 謝罪：「申し訳ございません。すぐにお持ちします。」
  4. テイクアウトはお客様に電話連絡し再提供
  5. 店長へ報告 → 再発防止を共有
✔ 商品受け渡し時は「2回確認」を徹底 ✔ テイクアウトはチェックリストの利用必須
4. お客様の体調不良
意識・アレルギー・酔いなど
▶
  1. 無理に動かさない
  2. 椅子・ベンチに座らせる or 安静に
  3. 意識が曖昧・ひどい場合 → 119番通報
  4. アレルギー疑い → 迷わず救急要請
  5. 店長へ連絡・対応引き継ぎ
✔ 迷ったら“救急車を呼ぶ”が基本 ✔ アレルギー反応は進行が早いため即対応
5. 店内でのケガ・転倒
床の水濡れ・段差など
▶
  1. 安全な場所へ誘導
  2. 応急処置（止血・冷却など）
  3. 必要なら救急要請
  4. 事故の原因（床濡れ・破損）をすぐ除去
  5. 店長へ報告 → 詳細を記録
✔ 同じ事故が2度起こらないよう原因を即改善
6. 迷惑客・トラブル客
泥酔・暴言・無銭飲食など
▶
【軽度（泥酔・大声）】
  1. トーンを落として落ち着いた接客
  2. 飲み物提供ストップ（店長判断）
  3. 必要なら退店をお願いする
【中度（暴言・他のお客様への迷惑）】
  1. スタッフだけで対応しない（複数で）
  2. 店長へ即エスカレーション
  3. 場合により警察相談（#9110）
【重度（暴力・器物破損）】
  1. 距離をとり安全確保
  2. 即 110 番
  3. 店長へ報告・指示を仰ぐ
✔ スタッフ1人で抱え込まない ✔ 周囲のお客様の安全を最優先
7. 火災・ガス漏れ
焼き台・厨房で最も危険な事故
▶
【火災発生】
  1. 初期消火（安全な場合のみ）
  2. 店内へ声掛け → 避難誘導
  3. 119番通報
  4. ガス元栓・電源OFF（可能な範囲で）
  5. 店長へ連絡
【ガス漏れ】
  1. 火気厳禁（コンロ・タバコNG）
  2. 換気 → 元栓OFF
  3. お客様避難
  4. ガス会社へ連絡
  5. 店長に報告
✔ パニックにさせず冷静な誘導が最重要 ✔ 火・電気を使わない
8. 停電・設備故障
POS・照明・厨房設備など
▶
  1. 状況確認（全体/一部か）
  2. お客様に状況説明
  3. 店長へ即報告
  4. 復旧までの提供可否を判断
  5. 危険がある場合は営業中断
✔ POSダウン時は手書き伝票へ切り替え ✔ 冷蔵設備故障は食品安全の観点から最優先で対応
