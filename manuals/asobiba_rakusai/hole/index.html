オペレーションマニュアル

接客・清掃・開店/閉店作業の基本手順まとめ

1. 接客の基本
挨拶・案内・注文・提供の流れ
▶
【入店対応】
  1. 3秒以内に「いらっしゃいませ！」
  2. 人数確認 → テーブルへご案内
  3. メニュー説明：「ご注文お決まりになりましたらお呼びください」
【注文対応】
  1. 必ず復唱（語尾までハッキリ）
  2. ハンディ入力ミスに注意
  3. アレルギー・提供時間のある商品は一言添える
【料理提供】
  1. 「〇〇になります。熱いのでお気をつけください」
  2. 提供した料理数を必ずテーブルで確認
  3. 空皿があれば下げる
【退店時】
  1. 席を立ったら「ありがとうございました！」
  2. すぐにテーブルリセット
✔ 声のトーン・笑顔・スピードが店の印象を決定 ✔ ハンディ入力ミスが最もクレームにつながりやすい
2. ホール巡回
気づき力で満足度UP
▶
  1. 空皿・グラス・串を見つけたらすぐ片付け
  2. 飲み物が少ないテーブルに「追加いかがですか？」
  3. 呼び出しには5秒以内に対応
  4. 床の汚れを見つけたら即清掃
  5. 焼台が忙しい時は提供を手伝う
✔ ホールは常に歩き回り状況把握する ✔ 無表情禁止。柔らかい表情をキープ
3. トイレ清掃
お店の印象が最も出る場所
▶
【清掃タイミング】
・開店前
・ピーク後（20–21時）
・閉店前 【手順】
  1. 換気ON、トイレットペーパー残量確認
  2. ゴミ箱を空にする
  3. 便座・便器を消毒スプレー→ペーパーで拭く
  4. 手洗い場の水滴をふく
  5. 床をモップ or シートで拭く
  6. 芳香剤チェック
✔ トイレの清潔感 = 店の評価 ✔ 汚れを見つけたら時間問わず即清掃
4. 開店準備
営業がスムーズに始まるために
▶
  1. レジ立ち上げ・釣り銭確認
  2. 店内の照明・音楽・空調チェック
  3. テーブルセット（箸 / おしぼり / 調味料）
  4. トイレチェック
  5. 焼台の火入れ・道具準備
  6. ドリンク補充（氷・ジョッキ・グラス）
✔ 開店前の準備がその日の回転効率を左右する ✔ 「誰が何を担当するか」を明確にしておく
5. 閉店作業
次の日の効率を最大化する
▶
  1. レジ締め → 売上確認 → レポート提出
  2. 客席・床掃除、ゴミ出し
  3. トイレ清掃（紙補充）
  4. 焼台の灰処理＆洗浄
  5. 冷蔵庫・食材の残量チェック
  6. 照明・電源OFF、戸締まり
✔ 次の日の仕込み量を最後にチェック ✔ 売上差異は必ずその日のうちに原因確認
6. 緊急時対応
クレーム・体調不良・事故など
▶
【クレーム】
  1. まずは遮らず最後まで聞く
  2. 「申し訳ございません」と謝罪
  3. 事実確認 → 解決策の提案
  4. 店長へ共有
【お客様の体調不良】
  1. 無理に動かさない
  2. 必要なら救急要請（119）
  3. 店長へ連絡
【ケガ・転倒】
  1. 安全確保 → 応急処置
  2. 事故原因（床濡れなど）を除去
  3. 店長へ報告 → 記録
【火災・ガス漏れ】
  1. 避難誘導 → 火気厳禁
  2. 119番通報
  3. ガス元栓OFF（可能範囲で）
  4. 店長へ連絡
✔ 強盗・暴力 → 身の安全が最優先、反抗しない ✔ クレーム → 言い訳禁止、必ず店長へエスカレーション
