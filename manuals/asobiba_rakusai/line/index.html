フードメニュー
ポテトフライ
一番人気の定番スナック
提供目安：7分
1皿：120g
油温：170〜180℃
脂がきつね色になり、表面がカリッとした状態がベスト。
材料（1皿）
冷凍ポテト120g（器に軽く山盛り）
塩ひとつまみ（0.5〜1g）
ケチャップ小さじ2（別皿）
手順
  1. フライヤーの油温を170〜180℃に安定させる。
  2. 冷凍ポテトを冷凍庫から出し、霜が多い場合は軽く振り落とす。
  3. バスケットにポテトを入れ、油に静かに入れる。
  4. 3〜4分揚げ、全体がきつね色になったらバスケットを上げる。
  5. よく油を切り、ステンバットの上で軽く塩をふる。
  6. 温めた器に盛り、ケチャップを別皿に入れて提供する。
チェックポイント
・一度に揚げる量が多いと油温が下がり、ベタつきの原因になります。
・揚げ置きは最大10分まで。時間が経ったものは必ず廃棄し、揚げ直しはしない。
鶏の唐揚げ
ジューシーで冷めても美味しい定番
提供目安：8分
1皿：5個（約100g）
油温：170℃ → 180℃
衣がカリッとし、中心までしっかり火が通っている状態。
材料（1皿）※仕込み済み唐揚げを使用
唐揚げ用鶏モモ（味付き）5個（約100g）
レモンカット1個
千切りキャベツひとつかみ（約20g）
マヨネーズ小さじ1（お好みで）
手順
  1. 仕込み済み唐揚げを冷蔵庫から出し、軽くほぐす。
  2. 170℃の油で4分揚げる。
  3. 一度バスケットを上げて30秒休ませ、油温を180℃に上げる。
  4. 再度1分30秒ほど二度揚げし、衣をカリッとさせる。
  5. キッチンペーパーの上で軽く油を切る。
  6. あらかじめキャベツを盛り付けた器に唐揚げを高く盛り、レモンとマヨネーズを添える。
中心温度の確認
・温度計がある場合は、中心温度75℃以上を確認。
・ない場合は、一つをカットし、赤みが完全になくなっているか確認する。
たこ焼き（冷凍）
おつまみに人気の定番メニュー
提供目安：6分
1皿：8個
表面に軽く焼き色が付き、中までしっかり温まっている状態。
材料（1皿）
冷凍たこ焼き8個
たこ焼きソース大さじ1.5
マヨネーズ大さじ1
青のりひとつまみ
かつお節ひとつまみ
手順（揚げ調理の場合）
  1. 冷凍たこ焼きを170℃の油で約3分揚げる。
  2. 表面が薄くきつね色になったらバスケットを上げ、油を切る。
  3. 温めた舟皿または丸皿にたこ焼きを並べる。
  4. ソース→マヨネーズ→青のり→かつお節の順にトッピングする。
レンジ調理バージョン
・店のオペレーションで電子レンジ指定がある場合は、そのワット数と秒数に合わせて温める。
・温め後、鉄板やオーブンで軽く焼き目を付けると食感が良くなる。
ソーセージ盛り合わせ
シェアしやすいおつまみ
提供目安：5分
1皿：4〜5本
焼き目が付き、表面がパリッとした状態。
材料（1皿）
ソーセージ（ミックス）4〜5本
ケチャップ小さじ2
粒マスタード小さじ1
手順
  1. 冷凍または冷蔵のソーセージを鉄板またはフライパンで中火加熱する。
  2. 転がしながら3〜4分焼き、全体に焼き色を付ける。
  3. 一口サイズの斜め切りにして器に盛る。
  4. ケチャップと粒マスタードを別添え、または器のふちに盛り付けて提供。
枝豆
スピード提供用おつまみ
提供目安：3分
1皿：100g
材料（1皿）
冷凍枝豆（塩味）100g
手順
  1. 指定があれば、袋の表示通りにレンジまたはボイルで解凍する。
  2. 水分が多い場合はキッチンペーパーで軽く拭き取る。
  3. 小鉢にこんもりと盛り付けて提供する。
ポイント
・解凍しすぎて熱くなりすぎないよう注意。
・提供前に塩味が薄い場合は、軽く塩をふる（入れすぎ注意）。

ハイボール
アルコールの定番メニュー
提供目安：3分
アルコール提供：年齢確認必須
飲酒運転・未成年飲酒禁止
氷たっぷり、グラスの外側は水滴を拭き取って提供。
材料（1杯）
ウイスキー30ml
炭酸水120ml
レモン（スライス）1枚（任意）
氷グラス9分目
手順
  1. ジョッキまたはグラスに氷をたっぷり入れる。
  2. ウイスキーを30ml注ぐ。
  3. 炭酸水を静かに注ぎ、マドラーで一度だけ軽く混ぜる。
  4. レモンを添え、外側を拭いて提供する。
注意事項
・必ず年齢確認を行う（身分証明書の確認）。
・泥酔状態のお客様には追加提供を控え、社員・責任者に報告する。
