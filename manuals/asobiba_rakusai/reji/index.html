レジマニュアル

会計ミス防止・操作統一のためのレジ業務マニュアル

1. 基本の会計の流れ
最も重要な基礎手順
▶
  1. お客様の利用内容（飲食 / 遊戯料金 / パック料金）を確認
  2. 伝票・ハンディ内容と POS の金額一致を確認
  3. 「本日のお会計 ○○円です」と読み上げる
  4. 支払い方法（現金 / カード / QR）を確認
  5. 決済後、レシートを渡し「ありがとうございました！」
✔「商品点数の入力漏れ」「遊戯時間の計算ミス」が最も多いミス ✔ 金額提示の前に **必ず内容を確認** してから会計へ進む
2. 現金会計
お釣り間違いを防ぐための基準手順
▶
  1. お客様から預かった金額を声に出して確認 （例：「5,000円お預かりします」）
  2. レジに正確に入力し、お釣り表示を確認
  3. お釣りは
    ①札 → ②小銭 → ③レシート の順番で、トレーに整えて渡す
  4. 「ありがとうございました」とアイコンタクト
画像
ポイント：
お釣りは指差し確認でミスを防止。
✔ 札を数え間違えないよう **声に出しながら確認** ✔ レジ入力前に財布へ札をしまわない
3. カード決済
IC・タッチ・磁気スワイプ対応
▶
  1. 金額を必ず読み上げてから決済端末へ入力
  2. お客様へ 「タッチ / 挿入 / スワイプ、どちらでも大丈夫です」 と案内
  3. 暗証番号 or サインが必要な場合はご案内
  4. レシート控えが必要か必ず確認
  5. 決済後、「ありがとうございました！」
✔ 決済エラー時は **エラー番号を必ずメモ** し店長へ報告 ✔ タッチ決済はカードを預からず、お客様に操作していただく
4. QRコード決済
PayPay・LINE Pay・楽天Pay など
▶
  1. QR決済の利用を確認
  2. レジで金額入力 → お客様へ QR 表示
  3. お客様のスマホで読み取り → 決済実行
  4. 必ず「決済完了画面」を目視確認
  5. レシートを渡して会計終了
✔ 完了画面を見ないまま操作すると **未決済のまま退店される恐れ** ✔ 店舗用QRの場合、金額入力ミスに注意
5. 割り勘・個別会計
最もミスが出やすい会計パターン
▶
  1. 最初に「均等割り」か「個別会計」か確認
  2. 均等割り → 人数を入力し、金額を確認してから案内
  3. 個別会計 → 1人分ずつ伝票を作り直し、会計
  4. 最後の1人で金額が合わないケースが多いので都度確認
✔ 個別会計が多い時は順番整理してお客様を誘導 ✔ 端数処理（1円〜2円）は店長判断に従う
6. レジ締め（閉店作業）
売上差異ゼロを目指す基本手順
▶
  1. 現金売上の合計をレジで確認
  2. レジ内の現金を数えて照合
  3. カード売上・QR売上の合計を確認
  4. レポートを印刷 or 写真保存
  5. 金庫へ保管し、店長へ売上報告
✔ レジ差異は、ほぼ「入力ミス」か「返金操作ミス」 ✔ 差異発生時 → 必ず時系列で追い、店長へ報告
